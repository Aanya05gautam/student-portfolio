# Student Portfolio Website

A simple and responsive student portfolio website built using React, Vite, and Tailwind CSS.  
The project demonstrates modern frontend development using component-based architecture and client-side routing.

## Features
- React-based UI
- Fast setup with Vite
- Tailwind CSS for styling
- Multi-page navigation using React Router
- Responsive design

## Tech Stack
React · Vite · Tailwind CSS · React Router · Git

## Run Locally
```bash
npm install
npm run dev

```
